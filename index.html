Machine Reading Comprehension for Tamil Language with Translated SQuAD
Oct 2025 – Mar 2026
Associated with Central Polytechnic College Chennai

Developed an Intelligent Machine Reading Comprehension (MRC) and Extractive Question Answering (QA) system tailored specifically for the Tamil language to solve context-based query extraction.

  • Model Architecture: Fine-tuned Google's MuRIL and deepset/xlm-roberta-base-squad2 for complex, agglutinative Tamil script.
  • Algorithmic Logic: Engineered a "Sliding Window Strategy" (384 tokens) to evaluate long-form documents without data loss.
  • Middleware & API: Architected a responsive Python Flask RESTful backend utilizing flask_cors for robust tokenization pipelines.
  • Frontend Interface: Designed a dynamic single-page dashboard with Tailwind CSS and statistical evaluation charts.
